# Financial Intelligence Dashboard

Bu proje, Armada Gıda için geliştirilmiş, yapay zeka destekli bir finansal analiz panosudur.

## Kurulum ve Çalıştırma

Bu proje statik HTML, CSS ve JavaScript kullanılarak geliştirilmiştir. Herhangi bir derleme (build) işlemine gerek yoktur.

1.  Bu klasördeki `index.html` dosyasını favori web tarayıcınızda (Chrome, Edge, vb.) açın.
2.  Sol menüden modüller arasında geçiş yapabilirsiniz.

## Özellikler

*   **Dashboard**: Piyasa özeti ve hızlı erişim kartları.
*   **Prediction (Tahmin)**: Yapay zeka destekli varlık fiyat tahmini (Mock Data).
*   **News Intelligence**: Küresel haber analizi ve sentiment skoru.
*   **Portfolio (Yapım Aşamasında)**: Portföy optimizasyonu.
*   **Comparison (Yapım Aşamasında)**: Varlık karşılaştırma.

## Teknoloji

*   HTML5
*   CSS3 (Modern Variables, Flexbox, Grid, Glassmorphism)
*   Vanilla JavaScript
*   Google Fonts (Inter, Outfit)
*   FontAwesome (İkonlar için)
